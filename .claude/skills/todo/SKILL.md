# /todo — 프로젝트 TODO 관리

프로젝트 루트의 `todo.md`를 읽고 현재 작업 상태를 파악한 뒤 작업을 이어서 진행한다.

## 실행 절차

1. `todo.md`를 읽어서 현재 상태를 파악한다
2. `[~]` 진행중 항목이 있으면 → 해당 작업을 바로 이어서 진행
3. 진행중 항목이 없으면 → 대기 목록을 보여주고 어떤 작업을 시작할지 사용자에게 물어본다
4. 작업 시작 시 해당 항목을 `[~]`로 변경하고, 세부 단계가 있으면 하위 체크박스 추가
5. 작업 완료 시 `[x]`로 변경하고 완료 섹션으로 이동

## 규칙

- **작업 시작 전** 반드시 `todo.md`의 해당 항목을 `[~]` 진행중으로 업데이트
- **작업 완료 시** 반드시 `[x]`로 변경 + 완료 날짜 기록
- **새 작업 발견 시** 대기 섹션에 `[ ]`로 추가
- **대규모 작업**은 별도 TODO 파일 생성 (예: `TODO-feature-name.md`) 후 `todo.md`에서 참조
- 상태 보고 시 텔레그램 채널이 연결되어 있으면 텔레그램으로도 보고
