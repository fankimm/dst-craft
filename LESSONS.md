## 코딩 패턴

### 매핑 데이터는 하드코딩보다 알고리즘으로
- **사례**: 국가 국기 이모지를 30개 하드코딩 → 새 국가(NZ, HU, UA) 누락
- **해결**: ISO 국가코드 2글자 → Regional Indicator 코드포인트 변환 함수
- **원칙**: 유한한 매핑이라도 규칙이 있으면 알고리즘으로 처리. 하드코딩은 누락과 유지보수 문제 유발

## SEO

### 글로벌 서비스는 영문 SEO 우선
- **사례**: `<html lang="ko">` 고정, 메타데이터 한국어 중심 → 영문 검색 노출 불리
- **해결**: lang="en" 기본 + 클라이언트 동적 감지, hreflang 태그, 영문 description 우선
- **원칙**: 글로벌 서비스는 영문 SEO를 기본으로 깔고, 로컬 언어를 보조로 추가
