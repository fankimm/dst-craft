# UI 용어집

프로젝트 전체에서 사용하는 UI 요소 명칭을 통일합니다.

| 용어 | 컴포넌트 | 설명 |
|------|----------|------|
| **아이템 카드** | `ItemIcon.tsx` | 그리드에 표시되는 아이템 이미지+이름 버튼 |
| **상세 패널** | `ItemDetail.tsx` | 아이템 카드 클릭 시 나타나는 레시피 정보 영역. 데스크톱: 하단 고정 패널, 모바일: 하단 Sheet(슬라이드업) |
| **재료 슬롯** | `MaterialSlot.tsx` | 상세 패널 안의 재료 아이콘+수량 표시 |
| **카테고리 그리드** | `CategoryGrid.tsx` | 홈 화면의 카테고리 선택 그리드 |
| **검색바** | `SearchBar.tsx` | 태그 기반 검색 입력 영역 |
| **검색 태그** | `SearchBar.tsx` 내 태그 칩 | 검색바 아래 표시되는 필터 태그 (캐릭터/카테고리/제작소/재료/아이템/텍스트) |
| **자동완성 드롭다운** | `SearchBar.tsx` 내 드롭다운 | 검색어 입력 시 나타나는 추천 목록 |
| **캐릭터 셀렉터** | `CharacterSelector.tsx` | 캐릭터 고유 카테고리 진입 시 캐릭터 선택 화면 |
| **브레드크럼** | `Breadcrumb.tsx` | 상단 네비게이션 경로 표시 (홈 > 카테고리 > ...) |
| **타이틀 바** | `AppShell.tsx` 내 Title bar | 최상단 앱 로고 + BETA 뱃지 표시 영역 |
| **요리솥 (탭)** | `CookpotApp.tsx` | 요리 시뮬레이터 탭 (재료 조합 테스트) |
| **추천 카테고리** | `CookingApp.tsx` | 요리 탭 홈의 체력/정신력/허기 기반 추천 음식 카테고리 타일 |
| **필터 칩** | `CookingApp.tsx` | 상세 패널 뱃지 클릭 시 레시피 목록 상단에 표시되는 제거 가능한 필터 표시 |
| **요리솥 시뮬레이터** | `CookpotApp.tsx` | 4개 재료를 넣어 결과 요리를 예측하는 시뮬레이터 |
| **재료 슬롯 (시뮬레이터)** | `CookpotApp.tsx` | 요리솥 시뮬레이터의 4개 재료 입력 칸 |
| **재료 선택기** | `IngredientPicker.tsx` | 재료 검색/선택 바텀시트 |
| **스테이션 토글** | `CookpotApp.tsx` | 요리솥/휴대용 요리솥 전환 버튼 |
| **릴리즈 노트 페이지** | `src/app/releases/page.tsx` | 버전별 변경 사항을 표시하는 독립 페이지. 설정에서 진입 |
| **즐겨찾기 타일** | `CategoryGrid.tsx`, `CookingApp.tsx` | 카테고리 그리드 상단의 즐겨찾기 카테고리 버튼 (하트 아이콘, 즐겨찾기 개수 표시) |
| **즐겨찾기 토글** | `ItemIcon.tsx`, `ItemDetail.tsx`, `RecipeCard`, `RecipeDetail` | 아이템 카드/상세 패널의 즐겨찾기 추가/제거 하트 버튼 |
| **리뷰 프롬프트** | `ReviewPrompt.tsx` | 5회 이상 방문 사용자에게 한 번만 표시되는 iOS 스타일 바텀시트. 별점 평가 + GitHub Star + 공유 기능 제공 |
| **체력 소모 표시** | `ItemDetail.tsx` | 상세 패널에 체력 소모가 있는 아이템의 HP 비용 표시 (빨간색 하트 아이콘) |
| **스킬트리 태그** | `ItemDetail.tsx` | 상세 패널에 스킬트리가 필요한 아이템의 보라색 스킬 아이콘 표시 |
| **스테이션 필요 표시** | `ItemDetail.tsx` | 상세 패널에 프로토타입 불가(nounlock) 아이템의 자물쇠 아이콘 표시 |
| **상태바** | (OS 영역) | iOS 상단의 노치/다이나믹 아일랜드 포함 영역 (`safe-area-inset-top`). 앱에서 직접 제어하지 않고 브라우저/OS에 위임 |
| **홈바** | (OS 영역) | iOS 하단의 홈 인디케이터 영역 (`safe-area-inset-bottom`). 하단 패널/푸터에 `pb-8` 여백으로 대응 |

| **피드백 섹션** | `SettingsPage.tsx` | 설정 페이지의 익명 의견 보내기 영역. 텍스트 입력 + 보내기 버튼 (1000자 제한, IP당 시간당 1회) |
| **스킬트리 시뮬레이터 (탭)** | `SkillSimulatorApp.tsx` | 캐릭터별 스킬트리를 Git Graph 스타일로 시각화하고 습득/해제를 시뮬레이션하는 탭 |
| **캐릭터 선택 그리드 (스킬)** | `SkillCharacterGrid.tsx` | 스킬 탭 홈에서 11개 캐릭터를 선택하는 그리드 |
| **스킬 노드 카드** | `SkillNodeCard.tsx` | 스킬트리 내 개별 스킬 표시. 아이콘+이름+습득 토글 버튼 |
| **스킬 레일** | `SkillRail.tsx` | Git Graph의 세로 라인. SVG로 그룹별 컬러 라인+정션 포인트 렌더링 |
| **스킬 게이트** | `SkillLockIndicator.tsx` | 잠금 조건 표시 인라인 디바이더. 다이아몬드(◆) + 조건 텍스트 |
| **스킬 상세 시트** | `SkillDetailSheet.tsx` | 스킬 노드 탭 시 열리는 바텀시트. 설명+해금 아이템+습득 버튼 |

## 버그 리포트 예시

> 도구 **카테고리 그리드** 클릭 → **잠자리채 아이템 카드** 클릭 → **상세 패널** 열림 → **밧줄 재료 슬롯** 클릭 → 상세 패널 닫힘 (밧줄 레시피 페이지로 이동해야 함)
